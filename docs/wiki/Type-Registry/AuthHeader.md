# AuthHeader

Constructed by `auth.bearer()`, `auth.basic()`, `auth.apiKey()`. Defined in
`Grob.Http`. Opaque type — not constructable directly.

| Member | Kind | Signature | Notes |
|--------|------|-----------|-------|
| `toString()` | method | `→ string` | Returns `"[AuthHeader]"` — never exposes credentials |

See also: [Grob.Http](../Plugins/Grob-Http.md)
